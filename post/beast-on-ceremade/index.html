Running BEAST 2 on CEREMADE server

Luke Kelly
Feb 24, 2021

From your directory on the cluster, download a tarball of BEAST 2 (v2.6.3 was the latest at the time of writing)

wget https://github.com/CompEvol/beast2/releases/download/v2.6.3/BEAST.v2.6.3.Linux.tgz

We can then extract the files to beast/ with

tar -xf BEAST.v2.6.3.Linux.tgz

Next, install all of the add-on packages

java -cp beast/lib/beast.jar beast.util.PackageManager -add Babel bacter BADTRIP BASTA bdmm BDSKY BEAST_CLASSIC BEASTLabs BEASTvntr Beasy besp bModelTest BREAK_AWAY CA CoalRe CodonSubstModels CoupledMCMC DENIM EpiInf FastRelaxedClockLogNormal feast FLC GEO_SPHERE Mascot MASTER MGSM MM MODEL_SELECTION MSBD MultiTypeTree NS ORC outercore PhyDyn phylodynamics PIQMEE PoMo SA SCOTTI SNAPP snapper SpeciesNetwork SSM STACEY StarBEAST2 substBMA TMA TreeStat2

We can then run BEAST 2 on an XML file, say example.xml by submitting the command

java -Xms64m -Xmx1024m -Djava.library.path=beast/lib/ -jar beast/lib/launcher.jar example.xml

to the queue. For example, you could have the following .pbs file

#!/bin/bash
#PBS -N <name of job>
#PBS -l walltime=24:00:00
#PBS -l nodes=1:ppn=1
#PBS -j oe
#PBS -M <user>@ceremade.dauphine.fr

java -Xms64m -Xmx1024m -Djava.library.path=beast/lib/ -jar beast/lib/launcher.jar stochastic-dollo.xml

We then submit the file to the TORQUE scheduler via

qsub stochastic-dollo.pbs

TORQUE will give you a job number and email you once the job is completed.
